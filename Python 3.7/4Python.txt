Функции


def - команда нужна для создания своей функции. Своя функция это повторение строчек кода которые в неё занесены

Пример своей функции:

def спавнпык():
    print('Хочыш спавнпык?')
    print('Хочыш спавнпык?')
    print('Хочыш спавнпык?')

Результатом будет при написании имя функции выведутся те строки которые занесены в неё:

Хочыш спавнпык?
Хочыш спавнпык?
Хочыш спавнпык?

Своя функция с принимающим аргументом
Пример:

def test(foo):
    print(foo * 6)

test(3)

В этом случаем сама функция уже прописана и нужно после своей функции внести параметр что бы произошло выполнение 
требования указанного в функции

Функция может возвращать значение с которыми были произведены операции
Пример:

def maximum(x, y):
    if x > y:
        return x
    else:
        return y

x = input('Введите первое число: ')
y = input('Введите второе число: ')

print(maximum(x,y))

Результатом выведения текста на экран будет большее из указаных чисел. 
Используется конструкция return(вернуть).
Когда происходит конструкция return то она возвращает и останавливает  дальнейшее исполнение.
В своей функции используются локальные переменные тоесть эти переменные можно использовать только в границах функции, за ёё пределами переменной не существует
