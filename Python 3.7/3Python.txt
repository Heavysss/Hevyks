range(Функция) - эта функция позволяет сделать диапазон чисел к примеру от 0 до 15
Пример:

foo = list(range(0,15))
print(foo)

Так же задействована функция list что бы преобразовать переменную foo в список от 0 до 15
Функция range принимает от 1 аргумента до 3 (эти 3 аргумента можно расшифровать как ОТ ДО ШАГ(подразумевает то на сколько нужно перескочить чисел))
Пример:

foo = list(range(1,11,2))
print(foo)

Результат:

foo = [0, 2, 4, 6, 8, 10]

ОБХОД СПИСКОВ

Можно делать обход списка тоесть посмотреть что есть в списке
Пример:

foo = [1,2,3,4,5]

i = 0

bar = len(foo) - 1 #Здесь отнимаем один индекс в результате чего программа сможет пройтись по всем индексам и не наткнутся на несуществующий индекс

while i <= bar:
    print(foo[i])
    i += 1

Результататом будет то что на экран выведутся всё что есть в списке всех типов данных

for (Цикл а не функция) - с помощью for можно сделать тот же самый обход
Пример:

foo = [1,2,3,4,5]

for x in foo:
    print(x)

Результат будет идентичный первому способу обхода списка

Цикл for можно использовать для повтора строк кода 
Пример:

for y in range(5):
    print('Hey')

Результат будет похож как с обходом тоесть каждый повтор будет использовать новую строчку в итоге получилось пять раз повторить строку 'Hey'
