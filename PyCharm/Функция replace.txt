Функция replace
С её помощью можно заменять одни строки на другие
Функция принимает два аргумента 
имя_переменной.replace('строка которую нужно заменять', 'На что заменить')

x = 'Самолёт падал прямо на север'

x = x.replace('падал','летел')

Также replace можно использовать для того что- бы заменить какое-то слово на пробел(То слово которое заменено превращается в пробел)

x = x.replace( 'прямо', ' ' )
